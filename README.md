#  Snake Game

# Project Overview:
This project is a graphical Snake Game developed using C++ and SFML library.
It includes real-time movement, collision detection, and score tracking.

# Features:
1. Real-time snake movement
2. Collision detection (self collision)
3. Dynamic score tracking
4. Food generation system
5. Game over handling

# Technologies Used:
1. C++
2. SFML (Simple and Fast Multimedia Library)
